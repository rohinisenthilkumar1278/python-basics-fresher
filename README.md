# Python Basics – Fresher Level

This repository contains beginner-level Python programs written as part of Python practice for freshers.  
The programs cover the following fundamental topics:

- **print()** – Displaying output, printing text, numbers, and patterns  
- **variables** – Storing and using data with proper variable names  
- **input()** – Taking input from the user (string, integer)  
- **type conversion** – Converting data between int, float, and string  

## Programs Included

1. `print_basics.py` – Examples of print statements and patterns  
2. `variables_demo.py` – Examples of variable declaration and usage  
3. `input_demo.py` – Examples of input and user interaction  
4. `type_conversion.py` – Examples of converting between different data types
